HTML é uma linguagem de marcação de tecnologia central da Internet, é utilizada para estruturar e apresentar conteúdo para a Internet, atualmente a linguagem está na quinta versão, ou seja, HTML5, a ultima revisão foi feita em 2014, desde então segue com o mesmo padrão, já a sua versão anterior foi padronizada em 1997 a HTML4.

Pontos Positivos da linguagem Html

  1. É uma linguagem que está disponível em diversas plataformas e sua performance é ótima em computadores, dispositivos móveis e tablets.
  2. É uma linguagem moderna e nova, já que teve uma revisão em 2014, mostrando ser atualizada.
  3. É uma linguagem de fácil compreensão sendo possível fazer edições rapidas e precisas.

Pontos Negativos da linguagem Html

  1. É uma linguagem que necessita ter um certo conhecimento para começar a utiliza-la, por mais que alguns aplicativos de edições de texto ajudem a com o alto preenchimento das tags, tem que saber qual é o uso correto das tags.
  2. É uma linguagem que precisa de uma base lógica de programação para melhor compreender e executa-la com êxito.
  3. É uma linguagem que é depende de outras para poder ter um documento com uma aparência que agrada a todos.
