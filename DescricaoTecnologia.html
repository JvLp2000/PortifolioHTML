Java é uma linguagem de programação orientada a objetos que começou a ser criada em 1991, na Sun MicroSystems. Teve seu inicio a partir do Green Project, que teve como seus mentores o Patrick Naughton, Mike Sheridan e James Gosling. Esse projeto não teve um intenção de criar uma nova linguagem de programação, mas sim antecipar e preparar a "proxima onda" que iria acontecer na informática, já que eles acreditavam que os aparelhos domesticos e os computadores teriam alguma ligação.

A primeira invenção dessa equipe foi a *7 (StarSeven) que foi um controle remoto de interface touchscreen, e esse aparelho tinha um mascote que ensinava o usuário utilizar o controle. Um dos desenvolvedores nomeou essa linguagem de Oak, o nome de uma árvore. Esse controle foi um antecessor da ideia de interação digital, que nós temos hoje em dia nos televisores digitais.

Nesta época, a internet estava ficando cada vez mais popular, e a equipe do Green Project começou a pensar em aplicações para o Oak na internet, onde a palavra chave é interação. Eles conseguiram adaptar a linguagem Oak para a internet, e em 1995 foi lançado o Java, que era uma versão atualizada do Oak para a internet.

Pontos Positivos da linguagem Java

  1. É uma linguagem de programação que é Multi-Threaded, ou seja, aceita multiplos processamentos.
  2. Tem uma Portabilidade muito boa, pois é possível rodar programas compilados em Web, celular, no relogio, dentre outros graças a máquina virtual JVM
  3. É uma linguagem orientada a objetos, o que ajuda na flexibilidade do códgio e flexibilidade e reusabilidade do código.

Pontos Negativos da linguagem Java

  1. A linguagem não possuir herença multipla.
  2. Ela tem uma performance reduzida.
  3. Ela possui um código prolixos, isso quer dizer que possui muitas palavras e muitas sentenças longas e complexas que são difícies de ler e entender, reduzindo a legibilidade do código.
