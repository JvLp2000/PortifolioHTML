Descrição Tecnologica

Nessa página eu gostaria de comentar sobre a linguagem de programção chamada Css.

Você pode acessar ao site que descreve mais sobre a linguagem por aqui Link CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML, é uma das principais linguagens da open web e é padronizada em navagadores web de acordo com as especificação da W3C. Desenvolvido em níveis, o CSS1 está atualmente obsoleto, o CSS2.1 é uma recomendação e o CSS3, agora dividido em pequenos módulos, está progredindo para a sua padronização.

Pontos Positivos da linguagem Css

  1. Ao usar o css, você pode deixar a página do estilo que quiser,apenas especificando uma vez, e depois chamando a variável de novo.
  2. Ajuda no processamento do site, fazendo com que ele gaste menos memória e fique mais rápido.
  3. É fácil de ser usado, consequentemente o desenvolvimento irá fluir mais.

Pontos Negativos da linguagem Css

  1. Nem sempre a linguagem irá funcionar em todos os browsers, assim sempre precisa testar a compatibilidade.
  2. O nível de segurança é baixo
  3. Para os iniciantes pode se tornar confuso por conta dos vários níveis de código.
