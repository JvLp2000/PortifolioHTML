Nessa página eu gostaria de comentar sobre a linguagem de programção chamada javascript.

Você pode acessar ao site oficial por aqui Link JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Pontos Positivos da linguagem javascript

  1. É uma otima linguagem para validação de formularios de entrada no lado do cliente, ja que o servidor nao precisa fazer nenhum processamento adicional
  2. Uma área de destaque do javascript é na criação de efeitos dinâmicos
  3. JavaScript é executado dentro do navegador do cliente, ele pode ser usado para alterar a aparência da tela do usuário após a página ter sido enviada pelo servidor. Isso permite criar alguns efeitos de imagem dinâmicos impressionantes.

Pontos Negativos da linguagem javascript

  1. O javaScript promete inchar severamente as páginas da web. O código JavaScript pode adicionar rapidamente centenas de linhas de código, se você estiver usando-o para fazer qualquer coisa, mesmo que remotamente interessante.
  2. Por causa da tendência do JavaScript de inchar as páginas da web, pode ser muito prejudicial para a facilidade de pesquisa do seu site
  3. É vulnerável brechas de segurança nos sistemas, navegadores e páginas envolvidas.
